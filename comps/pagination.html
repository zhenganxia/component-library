    # 分页

    TIP

    # 内置分页业务操作方法

    # 基本用法


    <template>
    <Pagination 
        :pageInfo="pageInfo"
        @pageChange="pageChange"
    />
</template>
<script>
  export default {
    data() {
      return {
        pageInfo:{
          total: 20,
          pageIndex: 1,
          pageSize: 10
        }
      }
    },
    methods: {
        pageChange(val) {
          console.log('当前分页情况',val)
        }
    }
  }
</script>

    # Attributes

    参数 说明 类型 可选值 默认值
    total 总数 number — —
    pageIndex 页码 number — 1

    # Events

    事件名 说明
    pageChange 更新页码重新赋值
